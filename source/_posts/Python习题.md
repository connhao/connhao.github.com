##### 2.写一个算法（流程图和python程序）：输入三个数，输出其最大者
```
def maxNum(a, b, c):
    if a > b:
        if a > c:
            print(a)
        else:
            print(c)
    else:
        if b > c:
            print(b)
        else:
            print(c)
```
